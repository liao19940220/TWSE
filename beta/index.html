📊 Excel 手動編輯與匯入規則說明

💡 當您匯出 Excel 備份後，可以直接手動在 Excel 內修改內容或新增列，再匯入系統。請務必遵守以下「交易紀錄」工作表之欄位填寫規則：

欄位名稱 (Excel) 填寫格式與範例 詳細說明
ID 留空（推薦） 手動新增交易時直接「留空」即可，系統匯入時會自動為其產生不重複 ID。
日期 2026-07-10 格式必須為 YYYY-MM-DD（半形橫線連結，如：2026-07-10）。
類型 BUY / SELL / DEPOSIT / WITHDRAW 必須填入對應的英文大寫指令：
• BUY (買入股票)
• SELL (賣出股票)
• DEPOSIT (帳戶入金)
• WITHDRAW (帳戶出金)
股票代號 2330 / CASH • 股票：填入股票代號（如：6488）
• 入金/出金：固定填入大寫 CASH
單價_金額 32900 / 950 • 股票：填入每股單價
• 入金/出金：請填入 "元"（在網頁輸入時雖輸入 32.9 千元，但在 Excel 內儲存時必須還原成實際的 32900）
數量 1000 / 1 • 股票：填入股數
• 入金/出金：固定填 1
手續費 0 / 20 該筆交易的手續費，沒有請填 0。
交易稅 0 / 15 該筆交易的交易稅（僅在賣出時需要），沒有請填 0。
